# 函館らーめん 汐のや — 西小山 ZINE Vol.01

東京・西小山駅前のラーメン店「函館らーめん 汐のや」のためのコンセプトウェブサイト。

> A website concept for **Hakodate Ramen Shionoya**, a small neighborhood ramen
> shop right outside Nishi-Koyama station, Shinagawa-ku, Tokyo (since 2016).
> Designed as a Showa-era shokudo zine: warm paper, sumi ink, beni red, broth gold.

## ファイル構成

```
shionoya/
├── index.html   ← マークアップ
├── styles.css   ← デザインシステム & 全スタイル
└── script.js    ← ナビ開閉、スクロール演出、丼パララックス
```

依存はGoogle Fonts（Reggae One, Kaisei Decol, Zen Kaku Gothic New, Yusei Magic, Fraunces）のみ。ビルド不要。

## ローカルでのプレビュー

```bash
cd shionoya
python3 -m http.server 8765
# → http://localhost:8765/
```

または `npx serve`、VS Code Live Server などでも可。

## 公開

そのままの状態で、GitHub Pages / Netlify / Vercel / Cloudflare Pages 等にデプロイできます。

## デザインコンセプト

**「街のごちそうZINE」** — 昭和の食堂ポスター × 食ジャーナル・エディトリアル。
モダンな雑誌の見開きデザインで、昭和レトロのラーメン屋らしい温かみと、
家族みんなで通える街の食堂の親しみやすさを表現。

### タイポグラフィ

- **Reggae One** — 昭和ポスター風の太い見出し
- **Kaisei Decol** — 装飾的な明朝（中見出し・引用）
- **Zen Kaku Gothic New** — 本文用の温かみあるゴシック
- **Yusei Magic** — 手書き風メモのアクセント

### モチーフ・演出

- 表紙（ヒーロー）— 大書きの「汐のや」 + ¥650 の朱色プライススタンプ
- 漫画雑誌風のフォリオ番号（01〜05）+ 朱の点リード
- ハーフトーン・ドットの紙テクスチャ（紙質感）
- マーキー型 ticker（情報を流す看板）
- 縦書き（tategaki）の小さな見出し
- 昭和の食堂SVGイラスト（のれん・カウンター・湯気）
- のれん風フッター（赤地に「函館らーめん 汐のや」）

## カスタマイズ

- **色**: `styles.css` 冒頭の `:root` 内のCSS変数を編集
- **フォント**: `:root` の `--f-poster` / `--f-display` / `--f-body` / `--f-hand` / `--f-latin` を差し替え
- **メニュー**: `index.html` の `.menu__list` 内の `<li>` を編集
- **電話番号 / 住所**: マストヘッド・カバー・ご案内・フッターの各箇所を更新
- **写真**: 店内・スープ・店主の写真があれば、各セクションの SVG プレースホルダーを `<img>` に差し替え可能

## 配慮した点

- **アクセシビリティ**:
  - `prefers-reduced-motion` 対応
  - 全装飾SVGに `aria-hidden`
  - フォーカスリング（朱色）
  - セマンティックなランドマーク（`header`/`nav`/`section`/`footer`）
- **モバイル**:
  - ハンバーガーメニュー（ドロップダウン）
  - グリッドが1カラムに再構成
  - フォントサイズ・余白を `clamp()` で流動的に
- **パフォーマンス**:
  - 外部画像ゼロ（全てインラインSVG）
  - JavaScript は最小限（マウス移動はrAFスロットル）
  - ビルド不要、依存はGoogle Fontsのみ
